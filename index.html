UDL 웹 접근성 도구

웹 접근성과 UDL(Universal Design for Learning) 원칙을 쉽게 확인하고 개선할 수 있는 도구

주요 기능

명도 대비 검사

텍스트와 배경색의 명도 대비를 검사하여 WCAG 2.1 기준을 충족하는지 확인합니다.

색맹 시뮬레이션

다양한 색맹 유형을 시뮬레이션하여 웹사이트의 색상 접근성을 테스트합니다.

UDL 원칙 분석

웹사이트가 UDL 원칙을 얼마나 잘 따르고 있는지 분석하고 개선점을 제안합니다.

테스트 영역

아래 영역에서 명도 대비 검사 기능을 테스트해볼 수 있습니다.

명도 대비 테스트 샘플

좋은 대비 (4.5:1 이상)
낮은 대비 (3:1 미만)
매우 낮은 대비 (2:1 미만)
접근성 도구 열기
